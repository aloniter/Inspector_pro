# Screenshot Plan

Apple accepts 1 to 10 screenshots per device size. For this iPhone-only app, capture the largest required iPhone size available in App Store Connect, then add other sizes only if App Store Connect requires them.

## Recommended Device

Use an iPhone Pro Max simulator/device size available on this Mac, preferably:

- iPhone 16 Pro Max, iOS 18.6, if using existing simulators
- iPhone 17 Pro Max, iOS 26.4.1, if you want the newest available simulator

## Screens to Capture

1. Login screen
   - Shows app logo, email/password fields, and professional sign-in flow.

2. Project list
   - Show at least one real-looking project.

3. Project detail / report list
   - Show a project with one or more inspection reports.

4. Report detail with photos
   - Show report metadata and a photo grid/list.

5. Photo annotation/editing
   - Show a photo with a simple annotation or marker.

6. Export options
   - Show PDF/DOCX export choice.

7. Branding/settings
   - Include only if the screen looks polished and has realistic company details.

## Content Rules

- Use non-sensitive sample inspection data.
- Do not show real customer names, private addresses, phone numbers, or emails.
- Use clean, realistic Hebrew content because Hebrew is the default experience.
- Keep the reviewer account password out of screenshots.
- Do not include payment, subscription, checkout, or external purchase messaging.

## Suggested Screenshot Order

1. Login
2. Projects
3. Reports
4. Report photos
5. Annotation
6. Export
7. Settings/branding

## Optional App Preview

Skip app preview video for the first submission unless screenshots are not enough. Screenshots are simpler and lower risk.
