## Description

Inspectley helps building inspectors and field professionals document issues, attach photos, add notes, and generate polished inspection reports for clients.

The workflow is built for field use: create a project, add a report, capture or import photos, annotate findings, write descriptions, and export a structured PDF or DOCX file.

Key features:

- Project and report management
- Camera capture and photo import
- Photo annotation
- Hebrew and RTL-friendly report text
- PDF and DOCX export
- Company branding in reports
- Hebrew and English interface support

An active account is required. Project and photo data are stored locally on the device. Supabase is used for authentication, company branding, and export authorization.
